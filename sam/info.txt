Guide de configuration Mosam
============================

1. Exportez vos clés dans les variables d'environnement:
   - `export OPENAI_API_KEY="..."` ou utilisez un coffre-fort.
   - `export MOSAM_MODEL="gpt-5-nano"`

2. Exemple de requête HTTP (résumé d'un classement CEDEAO):

curl https://api.openai.com/v1/responses \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${OPENAI_API_KEY}" \
  -d '{
    "model": "gpt-5-nano",
    "input": "Explique la classification CEDEAO pour un transformateur électrique 5kVA.",
    "store": false
  }'

3. Exemple Node.js

import OpenAI from "openai";

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

const response = await openai.responses.create({
  model: process.env.MOSAM_MODEL,
  input: "Donne la position tarifaire CEDEAO pour du beurre de karité brut.",
  store: false,
});
console.log(response.output_text);

4. Exemple Python

from openai import OpenAI
import os

client = OpenAI(api_key=os.environ["OPENAI_API_KEY"])

response = client.responses.create(
    model=os.environ["MOSAM_MODEL"],
    input="Rappelle les RGI applicables pour un ensemble mobilier en bois.",
    store=False,
)
print(response.output_text)
